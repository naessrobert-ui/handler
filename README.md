# Handler – Streamlit app

## Oppsett

1. Installer Python 3.10+
2. Åpne PowerShell
3. Gå til mappen:
   cd I:\\6\_EQUITIES\\Database\\"Python script"\\Handler
4. Lag virtuell miljø:
   python -m venv .venv
5. Aktiver:
   .\\.venv\\Scripts\\activate
6. Installer pakker:
   pip install -r requirements.txt
7. Start appen:
   streamlit run main.py
